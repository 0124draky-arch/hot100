## 数组与哈希表

### 核心思想
- **哈希表的本质**：用空间换时间，将查找时间从 O(n) 降到 O(1)
- **适用场景**：需要快速查找、计数、去重、判断存在性

### 常用模板

#### 1. 两数之和模式（查找配对）
```javascript
// 题目：找两个数使其和为target
function twoSum(nums, target) {
    const map = new Map(); // 存储 值 -> 索引
    for (let i = 0; i < nums.length; i++) {
        const complement = target - nums[i];
        if (map.has(complement)) {
            return [map.get(complement), i];
        }
        map.set(nums[i], i);
    }
}
```
**关键点**：
- 边遍历边查找，不需要两层循环
- 存储的是"需要的值"，而不是当前值

#### 2. 计数/频率统计模式
```javascript
// 统计元素出现次数
function countFrequency(arr) {
    const map = new Map();
    for (const item of arr) {
        map.set(item, (map.get(item) || 0) + 1);
    }
    return map;
}
```

#### 3. 前缀和 + 哈希表
```javascript
// 题目：和为K的子数组个数
function subarraySum(nums, k) {
    const map = new Map([[0, 1]]); // 前缀和 -> 出现次数
    let sum = 0, count = 0;
    for (const num of nums) {
        sum += num;
        if (map.has(sum - k)) {
            count += map.get(sum - k);
        }
        map.set(sum, (map.get(sum) || 0) + 1);
    }
    return count;
}
```
**关键点**：
- `sum - k` 表示"如果存在这个前缀和，那么中间部分就是k"
- 初始化 `map.set(0, 1)` 处理从头开始的子数组

### 典型题目
- 1. 两数之和 ⭐⭐⭐
- 49. 字母异位词分组 ⭐⭐
- 128. 最长连续序列 ⭐⭐⭐
- 560. 和为K的子数组 ⭐⭐⭐

### 易错点
❌ 忘记初始化前缀和的 `map.set(0, 1)`
❌ 哈希表的键类型问题（数组需要转字符串）
❌ 边遍历边修改哈希表时的逻辑顺序

## 双指针与滑动窗口

### 核心思想
- **双指针**：用两个指针代替嵌套循环，降低时间复杂度
- **滑动窗口**：维护一个动态区间，通过移动左右边界找到满足条件的子串/子数组

### 常用模板

#### 1. 对撞指针（两端向中间）
```javascript
// 适用于有序数组
function twoPointers(arr) {
    let left = 0, right = arr.length - 1;
    while (left < right) {
        // 根据条件移动指针
        if (满足条件) {
            // 处理逻辑
            left++;
            right--;
        } else if (需要增大) {
            left++;
        } else {
            right--;
        }
    }
}
```

#### 2. 快慢指针
```javascript
// 适用于链表、数组原地修改
function fastSlowPointers(arr) {
    let slow = 0;
    for (let fast = 0; fast < arr.length; fast++) {
        if (满足条件) {
            arr[slow] = arr[fast];
            slow++;
        }
    }
    return slow; // 新数组长度
}
```

#### 3. 滑动窗口（最重要！）
```javascript
// 通用滑动窗口模板
function slidingWindow(s) {
    const map = new Map(); // 窗口内元素统计
    let left = 0, right = 0;
    let result = 初始值;
    
    while (right < s.length) {
        // 1. 扩大窗口：right指针移动，加入新元素
        const c = s[right];
        right++;
        // 更新窗口数据
        map.set(c, (map.get(c) || 0) + 1);
        
        // 2. 判断是否需要收缩窗口
        while (窗口需要收缩的条件) {
            // 更新结果（在收缩前或收缩后，取决于题目）
            result = Math.max(result, right - left);
            
            // 缩小窗口：left指针移动，移除元素
            const d = s[left];
            left++;
            // 更新窗口数据
            map.set(d, map.get(d) - 1);
        }
        
        // 3. 更新结果（在这里或while内，取决于题目）
        result = Math.max(result, right - left);
    }
    return result;
}
```

**滑动窗口四步法：**
1. **右指针扩展**：加入新元素到窗口
2. **判断收缩**：窗口不满足条件时收缩
3. **左指针收缩**：移除左边元素
4. **更新答案**：在合适的时机更新结果

### 典型题目
- 3. 无重复字符的最长子串 ⭐⭐⭐（滑动窗口模板题）
- 76. 最小覆盖子串 ⭐⭐⭐（困难但重要）
- 438. 找到字符串中所有字母异位词 ⭐⭐
- 15. 三数之和 ⭐⭐⭐（对撞指针）
- 11. 盛最多水的容器 ⭐⭐（对撞指针）

### 易错点
❌ 滑动窗口的结果更新时机（收缩前还是收缩后？）
❌ 窗口长度计算：`right - left` 还是 `right - left + 1`？
❌ 边界条件：`while (left < right)` 还是 `while (left <= right)`？

## 链表

### 核心思想
- **虚拟头节点（哑节点）**：统一处理头节点，避免特殊判断
- **快慢指针**：找中点、判断环、找倒数第k个
- **反转链表**：迭代法需要保存前驱节点

### 常用模板

#### 1. 虚拟头节点
```javascript
function processLinkedList(head) {
    const dummy = new ListNode(0);
    dummy.next = head;
    let curr = dummy;
    
    while (curr.next) {
        // 处理逻辑
        curr = curr.next;
    }
    
    return dummy.next; // 返回真正的头节点
}
```

#### 2. 快慢指针找中点
```javascript
function findMiddle(head) {
    let slow = head, fast = head;
    // 奇数个节点：slow在中间
    // 偶数个节点：slow在中间偏右
    while (fast && fast.next) {
        slow = slow.next;
        fast = fast.next.next;
    }
    return slow;
}
```

#### 3. 反转链表（迭代）
```javascript
function reverseList(head) {
    let prev = null, curr = head;
    while (curr) {
        const next = curr.next; // 先保存下一个节点
        curr.next = prev;       // 反转指针
        prev = curr;            // prev前进
        curr = next;            // curr前进
    }
    return prev; // 新的头节点
}
```

### 典型题目
- 206. 反转链表 ⭐⭐⭐（必须掌握）
- 141. 环形链表 ⭐⭐
- 142. 环形链表II ⭐⭐⭐（快慢指针数学推导）
- 21. 合并两个有序链表 ⭐⭐
- 146. LRU缓存 ⭐⭐⭐（哈希表+双向链表）

### 易错点
❌ 反转链表时忘记保存 `next` 导致链表断裂
❌ 快慢指针的while条件：`fast && fast.next`
❌ 删除节点时忘记处理头节点的特殊情况（用哑节点解决）

## 栈与队列

### 核心思想
- **栈（Stack）**：后进先出（LIFO），用于处理"最近"相关的问题
- **单调栈**：栈内元素保持单调性，用于找"下一个更大/更小元素"

### 常用模板

#### 1. 括号匹配
```javascript
function isValid(s) {
    const stack = [];
    const map = { ')': '(', '}': '{', ']': '[' };
    
    for (const char of s) {
        if (char in map) {
            if (stack.pop() !== map[char]) return false;
        } else {
            stack.push(char);
        }
    }
    return stack.length === 0;
}
```

#### 2. 单调栈（下一个更大元素）
```javascript
// 题目：找每个元素右边第一个比它大的元素
function nextGreaterElement(nums) {
    const result = new Array(nums.length).fill(-1);
    const stack = []; // 存储索引
    
    for (let i = 0; i < nums.length; i++) {
        // 当前元素比栈顶大，说明找到了栈顶的"下一个更大元素"
        while (stack.length && nums[i] > nums[stack[stack.length - 1]]) {
            const idx = stack.pop();
            result[idx] = nums[i];
        }
        stack.push(i);
    }
    return result;
}
```

**单调栈关键点：**
- 栈内存储**索引**而不是值（方便记录位置）
- 维护**单调递减栈**找"下一个更大"
- 维护**单调递增栈**找"下一个更小"

### 典型题目
- 20. 有效的括号 ⭐⭐
- 739. 每日温度 ⭐⭐⭐（单调栈模板题）
- 84. 柱状图中最大的矩形 ⭐⭐⭐（单调栈困难题）
- 155. 最小栈 ⭐⭐

### 易错点
❌ 单调栈存值还是存索引？（通常存索引）
❌ while循环的条件判断
❌ 栈为空时的边界处理

## 二叉树

### 核心思想
- **递归三要素**：
  1. 确定递归函数的参数和返回值
  2. 确定终止条件
  3. 确定单层递归的逻辑
- **遍历 vs 分解问题**：
  - 遍历：在遍历过程中处理（前中后序）
  - 分解：将问题分解为子问题（递归返回值）

### 常用模板

#### 1. 递归遍历
```javascript
// 前序遍历：根 -> 左 -> 右
function preorder(root) {
    if (!root) return;
    处理当前节点(root);
    preorder(root.left);
    preorder(root.right);
}

// 中序遍历：左 -> 根 -> 右（BST中序是有序的！）
function inorder(root) {
    if (!root) return;
    inorder(root.left);
    处理当前节点(root);
    inorder(root.right);
}

// 后序遍历：左 -> 右 -> 根（适合自底向上）
function postorder(root) {
    if (!root) return;
    postorder(root.left);
    postorder(root.right);
    处理当前节点(root);
}
```

#### 2. 层序遍历（BFS）
```javascript
function levelOrder(root) {
    if (!root) return [];
    const result = [];
    const queue = [root];
    
    while (queue.length) {
        const levelSize = queue.length;
        const currentLevel = [];
        
        for (let i = 0; i < levelSize; i++) {
            const node = queue.shift();
            currentLevel.push(node.val);
            if (node.left) queue.push(node.left);
            if (node.right) queue.push(node.right);
        }
        result.push(currentLevel);
    }
    return result;
}
```

#### 3. 递归解决问题（返回值）
```javascript
// 例：计算树的最大深度
function maxDepth(root) {
    if (!root) return 0;
    const leftDepth = maxDepth(root.left);
    const rightDepth = maxDepth(root.right);
    return Math.max(leftDepth, rightDepth) + 1;
}
```

### 典型题目
- 94. 二叉树的中序遍历 ⭐⭐
- 102. 二叉树的层序遍历 ⭐⭐⭐
- 104. 二叉树的最大深度 ⭐⭐
- 226. 翻转二叉树 ⭐⭐
- 236. 二叉树的最近公共祖先 ⭐⭐⭐

### 易错点
❌ 递归终止条件写错（`!root` vs `!root.left && !root.right`）
❌ 前中后序遍历的顺序搞混
❌ 层序遍历忘记记录每层的节点数

## 动态规划

### 核心思想
- **动态规划 = 递归 + 记忆化**
- **五部曲**：
  1. 确定dp数组及下标含义
  2. 确定递推公式
  3. dp数组初始化
  4. 确定遍历顺序
  5. 举例推导dp数组

### 常用模板

#### 1. 一维DP
```javascript
// 例：爬楼梯
function climbStairs(n) {
    if (n <= 2) return n;
    const dp = new Array(n + 1);
    dp[1] = 1;
    dp[2] = 2;
    for (let i = 3; i <= n; i++) {
        dp[i] = dp[i-1] + dp[i-2];
    }
    return dp[n];
}
```

#### 2. 二维DP
```javascript
// 例：不同路径
function uniquePaths(m, n) {
    const dp = Array(m).fill(0).map(() => Array(n).fill(0));
    // 初始化第一行和第一列
    for (let i = 0; i < m; i++) dp[i][0] = 1;
    for (let j = 0; j < n; j++) dp[0][j] = 1;
    
    for (let i = 1; i < m; i++) {
        for (let j = 1; j < n; j++) {
            dp[i][j] = dp[i-1][j] + dp[i][j-1];
        }
    }
    return dp[m-1][n-1];
}
```

### 典型题目
- 70. 爬楼梯 ⭐⭐（DP入门）
- 198. 打家劫舍 ⭐⭐⭐
- 300. 最长递增子序列 ⭐⭐⭐
- 72. 编辑距离 ⭐⭐⭐（经典困难题）

### 易错点
❌ dp数组的初始化错误
❌ 遍历顺序错误（正序还是倒序？）
❌ 状态转移方程推导错误
